Beem SMS is lightweight SMS notification plugin for WordPress woocommerce stores.

== Description ==

Communicate with your customers via sms to improve sales and customer satisfaction.

== How does it work? ==

Beem SMS integrates with our SMS Api at [Beem](https://beem.africa/) to add support for text messaging.

Beem SMS allow you to send instant notifications to your woocommerce customers upon
receiving their orders, or updating them about the status of their orders.


* Prerequisites
1. Installation of WordPress
2. Installation of WooCommerce

== Installation ==

1. Extract the zip file
2. Copy the contents in the wp-content/plugins/ directory of your WordPress installation
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Generate your api key and secret key from your Beem dashboard. Visit [Beem](https://beem.africa/) for more details
